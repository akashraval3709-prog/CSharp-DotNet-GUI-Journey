# 🔒 Password Masking

This program demonstrates how to use a **one-dimensional character array** to mask a user's password.

## 📚 Concepts Covered

- One-Dimensional Array
- Character Array (`char[]`)
- String to Character Array
- Loops
- String Creation

## 🎯 Learning Outcome

- Convert a string into a character array.
- Replace characters with `*`.
- Create a new string from a character array.
