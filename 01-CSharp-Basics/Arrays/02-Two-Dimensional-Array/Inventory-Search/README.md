# 📦 Inventory Search

This program demonstrates the use of a **two-dimensional array** in C# to store product names and their locations, then search for a specific location.

## 📚 Concepts Covered

- Two-Dimensional Array
- Nested Loops
- String Comparison
- Array Traversal

## 🎯 Learning Outcome

- Store related data using a 2D array.
- Access array elements using row and column indexes.
- Search and display data from a two-dimensional array.
