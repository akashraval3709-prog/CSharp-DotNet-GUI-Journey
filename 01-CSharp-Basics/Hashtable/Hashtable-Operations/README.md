# 📚 Hashtable Operations in C#

This project demonstrates how to use the **Hashtable** collection in C#. It provides a simple menu-driven console application to perform common Hashtable operations such as adding, removing, displaying, clearing, and counting student records.

## 📖 Features

- Add Student Name
- Add Roll Number / Age
- Remove Data by Key
- Display All Records
- Clear Hashtable
- Count Total Records
- Menu-Driven Program

## 🛠 Concepts Covered

- Hashtable
- Key-Value Pairs
- Add()
- Remove()
- Clear()
- Count
- ContainsKey()
- DictionaryEntry
- foreach Loop
- switch Statement
- Infinite Loop (while)

## ▶️ Sample Menu

```text
===== HASHTABLE OPERATIONS MENU =====
1. Add Student Name
2. Add Roll Number / Age
3. Remove Detail
4. Display Details
5. Clear All Details
6. Get Total Count
7. Exit
```

## 🎯 Learning Objective

- Understand the Hashtable collection in C#
- Store and retrieve data using key-value pairs
- Practice CRUD-like operations on collections
- Build a menu-driven console application
