# 🗂️ Student Grades Using Hashtable

This program demonstrates the use of **Hashtable** in C# to store student names and their grades as key-value pairs.

## 📚 Concepts Covered

- Hashtable
- Key-Value Pairs
- Add()
- ContainsKey()
- Searching Data

## 🎯 Learning Outcome

- Learn how to use Hashtable.
- Store and retrieve data using keys.
- Search for values efficiently.
