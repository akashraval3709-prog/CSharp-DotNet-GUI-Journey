# 🔐 Password Validation System

A simple C# console application that validates a user's password with a maximum of **3 login attempts**.

## 📚 Concepts Covered

* While Loop
* Conditional Statements
* User Input
* Counters

## 🎯 Learning Outcome

* Using loops for repeated actions
* Handling user input
* Implementing basic authentication logic
