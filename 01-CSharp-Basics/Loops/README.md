# 🔄 Loops in C#

This folder contains a practical example of using **loops** in C# to build a simple password authentication system.

## 📚 Concepts Covered

* While Loop
* Conditional Statements (`if-else`)
* User Input (`Console.ReadLine()`)
* Variables
* Loop Control
* Authentication Logic

## 📂 Program

### 🔐 Password Validation System

This program:

* Allows the user to enter a password.
* Gives a maximum of **3 attempts**.
* Grants access when the correct password is entered.
* Displays an error message for incorrect passwords.
* Locks the account after 3 failed attempts.

## 🎯 Learning Outcome

By completing this program, I learned:

* How to use `while` loops effectively.
* How to control program flow using conditions.
* How to track user attempts with counters.
* How to build a simple login/authentication mechanism.
