# ➕ Sum and Average Calculator

This program demonstrates the use of a **while loop** in C# to calculate the sum and average of user-entered numbers.

## 📚 Concepts Covered

* While Loop
* User Input
* Variables
* Counters
* Sum Calculation
* Average Calculation

## 🎯 Learning Outcome

* Working with loops and counters
* Processing multiple user inputs
* Calculating sum and average dynamically
