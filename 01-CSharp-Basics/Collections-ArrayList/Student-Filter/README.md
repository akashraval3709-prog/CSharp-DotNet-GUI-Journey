# 📋 Student Filter Using ArrayList

This program demonstrates the use of **ArrayList** in C# by filtering student names that start with the letter **A**.

## 📚 Concepts Covered

- ArrayList
- Collections
- Loops
- String Methods (`StartsWith`)
- Filtering Data

## 🎯 Learning Outcome

- Learn how to use `ArrayList`.
- Store and iterate collection data.
- Filter items using string operations.
