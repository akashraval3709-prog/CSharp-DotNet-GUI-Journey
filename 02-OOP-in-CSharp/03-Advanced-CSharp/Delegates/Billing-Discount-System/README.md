# 💰 Billing Discount System using Delegates

This project demonstrates the use of **Delegates** in C# by applying different discount strategies to a customer's bill.

## 📖 Features

- Regular Customer (No Discount)
- Festival Discount (10% Off)
- VIP Member Discount (20% Off)
- Menu-Driven Console Application
- Calculates Final Payable Amount

## 🛠 Concepts Covered

- Delegates
- Method References
- Parameter Passing
- switch Statement
- Console Input / Output
- Static Methods
