# 📇 Student Indexer

This program demonstrates the use of **Indexers** in C# to store and access student roll numbers and names using array indexing.

## 📚 Concepts Covered

- Indexers
- Arrays
- Constructors
- Properties
- User Input

## 🎯 Learning Outcome

- Understand how indexers work in C#.
- Access objects using array-like syntax.
- Store and retrieve student information efficiently.
