# 💳 Digital Wallet

This program demonstrates **Encapsulation** by implementing a simple digital wallet system with PIN validation and balance management.

## 📚 Concepts Covered

* Encapsulation
* Private Fields
* Properties (`get` / `set`)
* Constructors
* Methods
* User Input
* Data Validation

## 🎯 Learning Outcome

* Protecting data using encapsulation
* Validating user input with properties
* Using constructors for object initialization
* Managing wallet balance securely
