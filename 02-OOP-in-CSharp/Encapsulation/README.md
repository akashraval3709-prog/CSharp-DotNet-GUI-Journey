# 🔒 Encapsulation in C#

This program demonstrates the concept of **Encapsulation** using a simple bank account example.

## 📚 Concepts Covered

* Private Fields
* Properties (`get` / `set`)
* Data Validation
* Object Creation

## 📂 Program

### 🏦 Bank Account

This program:

* Accepts account holder details.
* Validates age and balance.
* Prevents invalid data using properties.
* Displays account details after successful validation.

## 🎯 Learning Outcome

* Understanding Encapsulation
* Using private fields and properties
* Validating data with getters and setters
