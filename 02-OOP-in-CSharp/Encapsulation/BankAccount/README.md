# 🏦 Bank Account

This program demonstrates **Encapsulation** in C# using a simple bank account example.

## 📚 Concepts Covered

* Private Fields
* Properties (`get` / `set`)
* Data Validation
* Object Creation

## 🎯 Learning Outcome

* Understanding encapsulation
* Protecting data using private fields
* Validating user input with properties
* Applying OOP principles in a real-world example
