# 🏷️ Employee Properties

This program demonstrates the use of **Properties** in C# to manage employee information with basic validation.

## 📚 Concepts Covered

- Properties (`get` / `set`)
- Auto-Implemented Property
- Data Validation
- Object Creation
- User Input

## 🎯 Learning Outcome

- Understand how properties work in C#.
- Learn to validate data using property setters.
- Practice encapsulating class fields.
