# 🏗️ Constructors in C#

This folder contains examples demonstrating the use of **constructors** in C#. Constructors are special methods that initialize objects when they are created.

## 📚 Concepts Covered

* Default Constructor
* Parameterized Constructor
* Object Initialization
* Constructor Overloading

## 🎯 Learning Outcome

* Understanding how constructors work
* Initializing objects during creation
* Creating multiple constructors with different parameters
* Improving object-oriented programming skills
