# 👥 Customer Management using Static Members

This program demonstrates the use of **static variables** and **static properties** in C#.

## 📚 Concepts Covered

- Static Variables
- Static Properties
- Constructors
- Auto Increment ID
- Object Creation
- Arrays of Objects

## 🎯 Features

- Register multiple customers
- Generate unique Customer IDs automatically
- Display customer details
- Count total registered customers using a static member
