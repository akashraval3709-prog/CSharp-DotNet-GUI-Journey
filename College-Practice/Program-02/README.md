# 🔢 Number to Words Converter

This program converts a number between **1 and 999** into its corresponding English words.

## 📚 Concepts Covered

* Classes & Objects
* Methods
* Switch Statements
* Conditional Statements
* User Input

## 🎯 Learning Outcome

* Converting numbers into words
* Using methods for code organization
* Practicing conditional and switch-case logic
