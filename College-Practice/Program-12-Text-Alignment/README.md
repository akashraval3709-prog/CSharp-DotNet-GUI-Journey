# ↔️ Text Alignment Application

This Windows Forms application allows users to change the alignment of text inside a TextBox using **Left**, **Center**, and **Right** buttons.

## 📚 Concepts Covered

- Windows Forms
- TextBox
- Buttons
- Click Events
- Text Alignment

## 🎯 Learning Outcome

- Handling button click events
- Changing TextBox properties dynamically
- Building a simple interactive Windows Forms application
