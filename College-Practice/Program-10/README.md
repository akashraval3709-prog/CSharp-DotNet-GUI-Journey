# 📊 Jagged Array

This program demonstrates the use of a **Jagged Array** in C# to store and display different grades for multiple students.

## 📚 Concepts Covered

- Jagged Arrays
- Nested Loops
- User Input
- Array Traversal

## 🎯 Learning Outcome

- Creating jagged arrays
- Storing different numbers of grades
- Displaying array elements using nested loops
