# 🖥️ Windows Forms Basics

This project demonstrates the fundamentals of **Windows Forms** in C#. It includes basic user interactions such as changing the form background color, displaying a message box, and clearing user input from a TextBox.

## 📚 Features

- Change Form Background Color
- Display MessageBox
- Clear TextBox
- Set Focus to TextBox
- Basic Button Click Events
- Windows Forms Event Handling

## ▶️ How It Works

1. Enter your name in the TextBox.
2. Click **Change Color** to change the form background.
3. Click **Show Message** to display a MessageBox.
4. Click **Clear** to clear the TextBox and focus the cursor.

## 🎯 Learning Objectives

- Understand Windows Forms controls
- Handle Button Click events
- Use MessageBox
- Change Form properties at runtime
- Work with TextBox methods (`Clear()`, `Focus()`)
