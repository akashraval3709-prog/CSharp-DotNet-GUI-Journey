# 📐 Constructor & Method Overloading

This program demonstrates **constructor overloading** and **method overloading** by calculating the area of a rectangle and a circle.

## 📚 Concepts Covered

- Constructors
- Constructor Overloading
- Method Overloading
- Classes & Objects
- User Input

## 🎯 Learning Outcome

- Initializing objects using different constructors
- Using overloaded methods with different parameters
- Calculating the area of different shapes
