# 📝 Registration Form

This Windows Forms application allows users to enter their first name, last name, and email address. It validates the input fields and displays the entered information in a message box upon successful registration.

## 📚 Concepts Covered

* Windows Forms
* Labels
* TextBoxes
* Button Click Event
* MessageBox
* Input Validation

## 🎯 Learning Outcome

* Designing a simple Windows Form
* Handling button click events
* Validating user input
* Displaying information using a MessageBox
