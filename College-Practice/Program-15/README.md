# 🌍 Country Selection (ComboBox)

## 📄 Files

```text
Program.cs

```

## 💡 Logic

- Create a Windows Forms application.
- Add a ComboBox containing a list of countries.
- Handle the `SelectedIndexChanged` event.
- Display the selected country inside a Label.
- Show a default message before any country is selected.

- Windows Forms

- SelectedIndexChanged Event
- GUI Programming
