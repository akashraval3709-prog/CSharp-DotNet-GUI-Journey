# 🧮 Delegate Operations

## 📄 File Name

```text
Program.cs
```

## 💡 Logic

- Create a delegate named `operation`.
- Create a class `ArithmeticOperations`.
- Implement two methods:
  - Addition
  - Multiplication
- Assign methods to the delegate based on user choice.
- Invoke the selected method through the delegate.
- Display the result.

- Delegates
- Methods
- Classes & Objects
- User Input
- Switch Statement
- Delegate Invocation
