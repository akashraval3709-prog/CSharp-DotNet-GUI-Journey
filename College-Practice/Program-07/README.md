# 🔤 String Manipulation

This program demonstrates various string manipulation methods in C# by taking user input and performing different operations.

- ToUpper()
- ToLower()
- Concat()
- Substring()
- Insert()
- Remove()
- IndexOf()

- Performing common string operations
- Working with built-in string methods
- Understanding string manipulation in C#
